### 1. Core AI/Cognitive Layer
- **skintwin**: Alchemist Engine - tensor transformation pipelines
- **skintwin-asi**: Dual-hemisphere cognitive architecture (Deep Tree Echo + Marduk)
- **multiskin**: Biological modeling with JAX neural computation

### 2. Business Applications
- **skintwin-customer-portal**: Full-stack multi-tenant platform
- **regima-training-lms**: Learning management system

### 3. Integration Layer
- **skintwin-integrations**: External platform connectors
- **org-skin**: GitHub organization management

### 4. Frontend Templates
- **business-directory-template**: Directory UI template
- **bus-listing**: Business listing variant
